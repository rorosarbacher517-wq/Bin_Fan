# Transmission Icing Operation Guidelines Demo Corpus

## High icing risk

If tower-level icing risk is high, operators should verify nearby observations,
increase patrol priority, prepare de-icing or anti-icing resources, and watch
critical spans with high elevation or windward exposure.

## Dynamic line rating watch

If dynamic line rating margin is low, operators should check current loading,
evaluate transfer capability, and avoid relying on weather-driven headroom
until refreshed forecasts confirm stable cooling conditions.

## Agent response discipline

The Agent must cite weather triggers, terrain or remote-sensing factors, model
score, and DLR margin. If data is missing, it should say which data is missing
and return a conservative fallback.
